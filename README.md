# Python-Based Quantitative Trading Strategies
I built this collection of trading strategies and technical analysis indicators using Python. This repository is like my library where I store my research models and show off my quantitative development skills.

## Project. Timeline
* I started researching and developing this project in September 2025.
* I moved it to GitHub. Deployed it in June 2026.
* Now its archived for demonstrating my portfolio and I'm still optimizing it.

## Core. Features
* I implemented custom indicators like Moving Averages, RSI, MACD and Bollinger Bands.
* I created backtesting scripts to evaluate how my strategies would have performed in the past.
* I integrated data processing with financial data APIs to fetch, parse and structure historical market data.
